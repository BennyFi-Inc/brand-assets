## App Themes

### PledgePool (Participant App)
- **Primary**: `#82C663` (brighter green)
- **Secondary**: `#EBA128` (warmer amber)
- **Dark**: `#263539` (deeper charcoal)
- **Theme feel**: Fresh, inviting, growth-oriented

### Composer (Admin App)
- **Primary**: `#3A4447` (dark charcoal — professional admin feel)
- **Secondary**: `#89C66A` (green accent)
- **Accent**: `#EAA33A` (amber for highlights)
- **Theme feel**: Professional, authoritative, clean

## Semantic Colors
- Positive/Success: `#4CAF50`
- Negative/Error: `#F44336`
- Info: `#2196F3`
- Warning: `#EAA33A` (brand amber)

## Neutrals
- White: `#FFFFFF`
- Light bg: `#F5F7F3`
- Grey scale: `#F0F2EE` → `#1A1C19`

## Typography
- Body: Roboto (system sans-serif stack fallback)
- Monospace: Roboto Mono / SF Mono / Fira Code
- Wordmark: Custom rounded sans-serif (logo only, not for body text)
